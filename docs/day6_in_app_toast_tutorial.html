Day 6 Tutorial

Flutter 6일차: 앱 내부 토스트 시스템 붙이기

이 문서는 6일차에 진행한 app_toast.dart 추가와 실제 모드/데모 모드에 토스트를 연결하는 작업을 튜토리얼 방식으로 정리한 문서입니다.

1단계

공통 토스트 파일을 새로 만듭니다.

2단계

OverlayEntry로 앱 위에 뜨는 토스트 UI를 만듭니다.

3단계

듣는 중 / 처리 중 / 완료 상태를 enum으로 나눕니다.

4단계

실제 모드에 토스트를 연결합니다.

5단계

데모 모드에도 같은 토스트 흐름을 붙입니다.

오늘의 목표

  • 사용자가 버튼을 눌렀을 때 앱이 즉시 반응해야 합니다.
  • 듣기 시작, 처리 중, 완료 상태가 눈에 보여야 합니다.
  • 실제 모드와 데모 모드가 같은 알림 경험을 가져야 합니다.
  • OS 전용 팝업 없이도 충분히 자연스러운 피드백을 제공해야 합니다.

핵심 개념

OverlayEntry

현재 화면 위에 잠깐 뜨는 커스텀 UI를 올리는 방식입니다.

Overlay.of(context)

토스트를 실제로 꽂아 넣을 앱 오버레이 레이어를 찾습니다.

Future/Timer

일정 시간 후 토스트를 자동으로 닫게 합니다.

공통 함수화

showAppToast() 하나만 호출하면 어느 화면에서든 같은 토스트를 띄울 수 있습니다.

상태 enum

듣는 중, 처리 중, 완료를 문자열이 아닌 enum으로 관리해 분기를 깔끔하게 합니다.

1. app_toast.dart 만들기

lib/features/notifications/presentation/app_toast.dart

6일차의 핵심은 앱 전용 토스트 시스템을 공통 파일로 만드는 것입니다. 이렇게 해야 실제 모드와 데모 모드가 같은 알림 시스템을 공유할 수 있습니다.

핵심 생각: 토스트는 특정 화면 전용 UI가 아니라, 앱 전체에서 재사용하는 공통 피드백 시스템이어야 합니다.

2. 왜 별도 토스트를 만드는가

기본 SnackBar를 쓸 수도 있지만, 오늘은 앱 분위기에 더 잘 맞는 전용 토스트 UI를 만드는 것이 목적입니다.

  • 위치 제어가 더 자유롭습니다.
  • 상태별 색상과 아이콘을 더 자연스럽게 줄 수 있습니다.
  • 실제 모드와 데모 모드가 같은 디자인 시스템을 쓰게 됩니다.

3. 실제 모드와 데모 모드에 연결하기

토스트는 아래 시점들에 연결합니다.

  • 듣기 시작: AppToastState.listening
  • 처리 시작: AppToastState.processing
  • 작업 완료: AppToastState.success

실제 모드와 데모 모드가 모두 같은 방식으로 반응해야 앱이 하나의 제품처럼 느껴집니다.

최종 코드 1: app_toast.dart

lib/features/notifications/presentation/app_toast.dart
import 'dart:async';

import 'package:flutter/material.dart';

enum AppToastState {
  info,
  listening,
  processing,
  success,
}

void showAppToast(
  BuildContext context,
  String message, {
  String title = 'Navi: Voice Navigator',
  AppToastState state = AppToastState.info,
  int durationMs = 2200,
}) {
  _ToastOverlay.instance.show(
    context,
    title: title,
    message: message,
    state: state,
    durationMs: durationMs,
  );
}

class _ToastOverlay {
  _ToastOverlay._();

  static final _ToastOverlay instance = _ToastOverlay._();

  OverlayEntry? _entry;
  Timer? _dismissTimer;

  void show(
    BuildContext context, {
    required String title,
    required String message,
    required AppToastState state,
    required int durationMs,
  }) {
    _dismissTimer?.cancel();
    _entry?.remove();

    final overlay = Overlay.of(context);
    _entry = OverlayEntry(
      builder: (context) {
        return _ToastView(
          title: title,
          message: message,
          state: state,
        );
      },
    );

    overlay.insert(_entry!);

    _dismissTimer = Timer(Duration(milliseconds: durationMs), () {
      _entry?.remove();
      _entry = null;
    });
  }
}

class _ToastView extends StatelessWidget {
  const _ToastView({
    required this.title,
    required this.message,
    required this.state,
  });

  final String title;
  final String message;
  final AppToastState state;

  @override
  Widget build(BuildContext context) {
    final colors = _resolveColors(state);
    final icon = _resolveIcon(state);

    return Positioned(
      top: 24,
      right: 24,
      child: Material(
        color: Colors.transparent,
        child: Container(
          width: 320,
          padding: const EdgeInsets.all(16),
          decoration: BoxDecoration(
            color: colors.background,
            borderRadius: BorderRadius.circular(18),
            border: Border.all(color: colors.border),
            boxShadow: const [
              BoxShadow(
                color: Color(0x22000000),
                blurRadius: 18,
                offset: Offset(0, 10),
              ),
            ],
          ),
          child: Row(
            crossAxisAlignment: CrossAxisAlignment.start,
            children: [
              Container(
                width: 40,
                height: 40,
                decoration: BoxDecoration(
                  color: colors.iconBackground,
                  borderRadius: BorderRadius.circular(12),
                ),
                child: Icon(
                  icon,
                  color: colors.iconColor,
                  size: 20,
                ),
              ),
              const SizedBox(width: 12),
              Expanded(
                child: Column(
                  crossAxisAlignment: CrossAxisAlignment.start,
                  children: [
                    Text(
                      title,
                      style: const TextStyle(
                        fontSize: 14,
                        fontWeight: FontWeight.w800,
                      ),
                    ),
                    const SizedBox(height: 4),
                    Text(
                      message,
                      style: const TextStyle(
                        fontSize: 13,
                        height: 1.45,
                        color: Color(0xFF475569),
                      ),
                    ),
                  ],
                ),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

IconData _resolveIcon(AppToastState state) {
  switch (state) {
    case AppToastState.info:
      return Icons.info_outline_rounded;
    case AppToastState.listening:
      return Icons.mic_rounded;
    case AppToastState.processing:
      return Icons.hourglass_top_rounded;
    case AppToastState.success:
      return Icons.check_rounded;
  }
}

_ToastColors _resolveColors(AppToastState state) {
  switch (state) {
    case AppToastState.info:
      return const _ToastColors(
        background: Colors.white,
        border: Color(0xFFE2E8F0),
        iconBackground: Color(0xFFEFF6FF),
        iconColor: Color(0xFF2563EB),
      );
    case AppToastState.listening:
      return const _ToastColors(
        background: Colors.white,
        border: Color(0xFFBFDBFE),
        iconBackground: Color(0xFFEFF6FF),
        iconColor: Color(0xFF2563EB),
      );
    case AppToastState.processing:
      return const _ToastColors(
        background: Colors.white,
        border: Color(0xFFFDE68A),
        iconBackground: Color(0xFFFEF3C7),
        iconColor: Color(0xFFF59E0B),
      );
    case AppToastState.success:
      return const _ToastColors(
        background: Colors.white,
        border: Color(0xFFBBF7D0),
        iconBackground: Color(0xFFDCFCE7),
        iconColor: Color(0xFF16A34A),
      );
  }
}

class _ToastColors {
  const _ToastColors({
    required this.background,
    required this.border,
    required this.iconBackground,
    required this.iconColor,
  });

  final Color background;
  final Color border;
  final Color iconBackground;
  final Color iconColor;
}

6일차 완료 기준

화면 기준

  • 앱 내부 토스트가 화면 위에 자연스럽게 뜹니다.
  • 듣기 / 처리 / 완료 상태가 색과 아이콘으로 구분됩니다.

동작 기준

  • 실제 모드에 토스트가 연결됩니다.
  • 데모 모드에도 같은 토스트 흐름이 연결됩니다.
  • 토스트는 일정 시간 뒤 자동으로 사라집니다.

학습 기준

  • OverlayEntry가 왜 필요한지 설명할 수 있습니다.
  • 공통 함수 showAppToast()의 장점을 이해합니다.
  • 상태 enum으로 UI를 나누는 이유를 설명할 수 있습니다.
