1. settings_controller.dart 수정

lib/features/settings/application/settings_controller.dart

오늘부터 실제 모드 설정 모달도 내부에서 임시 draft 설정을 들고 있습니다. 그래서 저장 버튼을 눌렀을 때 설정 전체를 한 번에 반영할 메서드가 필요합니다.

어디에 추가하나
Future<void> save() 메서드 바로 아래에 추가하면 가장 보기 좋습니다.
void replaceAll(AppSettings next) {
  state = next;
}

이 메서드는 저장 시점에 draft 설정을 통째로 현재 설정으로 바꾸는 역할입니다.

2. 실제 모드 설정 모달 전면 교체

lib/features/settings/presentation/settings_modal.dart

이 단계가 오늘의 핵심입니다. 기존 실제 모드 설정 모달은 탭형이긴 하지만 아직 TextField로 단축키를 입력받고 있었습니다. 이제는 Day 7 데모 모드처럼 draft 상태 + 단축키 직접 캡처 + 저장 시 일괄 반영 구조로 바꿉니다.

왜 전체 교체하나요?
기존 구조 위에 조금씩 덧붙이는 것보다, 데모 모드에서 이미 검증된 구조로 다시 맞추는 것이 훨씬 안전합니다. 특히 포커스 처리와 단축키 캡처는 부분 수정보다 전체 교체가 오류를 줄입니다.
진행 방법
기존 settings_modal.dart 파일 내용을 전부 지우고, 아래 전체 코드를 그대로 넣습니다.

class SettingsModal extends ConsumerStatefulWidget {
  const SettingsModal({
    super.key,
    required this.onClose,
  });

  final VoidCallback onClose;

  @override
  ConsumerState<SettingsModal> createState() => _SettingsModalState();
}

class _SettingsModalState extends ConsumerState<SettingsModal> {
  static const _tabs = ['기본 설정', '단축키', '보안', '화면 설정'];

  final FocusNode _focusNode = FocusNode(debugLabel: 'settings_modal_focus');
  final ScrollController _bodyScrollController = ScrollController();

  late AppSettings _draftSettings;
  int _selectedTab = 0;
  String? _capturingField;

  @override
  void initState() {
    super.initState();
    _draftSettings = ref.read(settingsControllerProvider);

    WidgetsBinding.instance.addPostFrameCallback((_) {
      if (mounted) {
        _focusNode.requestFocus();
      }
    });
  }

  @override
  void dispose() {
    _bodyScrollController.dispose();
    _focusNode.dispose();
    super.dispose();
  }

  KeyEventResult _handleKeyEvent(FocusNode node, KeyEvent event) {
    if (event is! KeyDownEvent) {
      return KeyEventResult.ignored;
    }

    if (_capturingField != null) {
      if (event.logicalKey == LogicalKeyboardKey.escape) {
        setState(() => _capturingField = null);
        return KeyEventResult.handled;
      }

      if (event.logicalKey == LogicalKeyboardKey.backspace ||
          event.logicalKey == LogicalKeyboardKey.delete) {
        _setShortcut(_capturingField!, '');
        setState(() => _capturingField = null);
        return KeyEventResult.handled;
      }

      final shortcut = ShortcutUtils.captureFromEvent(event);
      if (shortcut == null || shortcut.isEmpty) {
        return KeyEventResult.handled;
      }

      _setShortcut(_capturingField!, shortcut);
      setState(() => _capturingField = null);
      return KeyEventResult.handled;
    }

    return KeyEventResult.ignored;
  }

  void _update(AppSettings next) {
    setState(() => _draftSettings = next);
  }

  void _selectTab(int value) {
    setState(() => _selectedTab = value);
    if (_bodyScrollController.hasClients) {
      _bodyScrollController.jumpTo(0);
    }
  }

  void _setShortcut(String field, String value) {
    final normalized = ShortcutUtils.normalize(value);
    var shortcuts = _draftSettings.shortcuts;

    if (normalized.isNotEmpty) {
      if (field != 'listen' &&
          ShortcutUtils.normalize(shortcuts.listenToggle) == normalized) {
        shortcuts = shortcuts.copyWith(listenToggle: '');
      }
      if (field != 'screen' &&
          ShortcutUtils.normalize(shortcuts.screenRead) == normalized) {
        shortcuts = shortcuts.copyWith(screenRead: '');
      }
      if (field != 'settings' &&
          ShortcutUtils.normalize(shortcuts.openSettings) == normalized) {
        shortcuts = shortcuts.copyWith(openSettings: '');
      }
    }

    switch (field) {
      case 'listen':
        shortcuts = shortcuts.copyWith(listenToggle: normalized);
        break;
      case 'screen':
        shortcuts = shortcuts.copyWith(screenRead: normalized);
        break;
      case 'settings':
        shortcuts = shortcuts.copyWith(openSettings: normalized);
        break;
    }

    _update(_draftSettings.copyWith(shortcuts: shortcuts));
  }

  Future<void> _saveDraft() async {
    final notifier = ref.read(settingsControllerProvider.notifier);
    notifier.replaceAll(_draftSettings);
    await notifier.save();
    if (mounted) {
      widget.onClose();
    }
  }

  @override
  Widget build(BuildContext context) {
    final surfaceTheme = Theme.of(context).extension<AppSurfaceTheme>()!;

    return Focus(
      focusNode: _focusNode,
      autofocus: true,
      onKeyEvent: _handleKeyEvent,
      child: Center(
        child: Container(
          width: 780,
          height: 620,
          decoration: BoxDecoration(
            color: surfaceTheme.surface,
            borderRadius: BorderRadius.circular(24),
            border: Border.all(color: surfaceTheme.border),
            boxShadow: const [
              BoxShadow(
                color: Color(0x33000000),
                blurRadius: 28,
                offset: Offset(0, 14),
              ),
            ],
          ),
          child: Column(
            children: [
              _ModalHeader(
                title: '설정',
                subtitle: '실제 모드 설정과 화면 표시 옵션을 조정합니다.',
                onClose: widget.onClose,
              ),
              Divider(height: 1, color: surfaceTheme.border),
              Expanded(
                child: Row(
                  crossAxisAlignment: CrossAxisAlignment.stretch,
                  children: [
                    _Sidebar(
                      items: _tabs,
                      selectedTab: _selectedTab,
                      onSelect: _selectTab,
                    ),
                    Container(width: 1, color: surfaceTheme.border),
                    Expanded(
                      child: LayoutBuilder(
                        builder: (context, constraints) {
                          return SingleChildScrollView(
                            controller: _bodyScrollController,
                            padding: const EdgeInsets.all(20),
                            child: Align(
                              alignment: Alignment.topLeft,
                              child: ConstrainedBox(
                                constraints: BoxConstraints(
                                  minWidth: constraints.maxWidth,
                                ),
                                child: _buildBody(),
                              ),
                            ),
                          );
                        },
                      ),
                    ),
                  ],
                ),
              ),
              Container(
                padding: const EdgeInsets.fromLTRB(20, 16, 20, 20),
                decoration: BoxDecoration(
                  color: surfaceTheme.contentBackground,
                  border: Border(top: BorderSide(color: surfaceTheme.border)),
                  borderRadius:
                      const BorderRadius.vertical(bottom: Radius.circular(24)),
                ),
                child: Row(
                  children: [
                    Text(
                      _capturingField == null
                          ? '저장 전에는 임시 draft 설정만 변경됩니다.'
                          : '원하는 키를 누르세요. Delete는 해제, Esc는 취소입니다.',
                      style: TextStyle(
                        fontFamily: 'Pretendard',
                        fontSize: 12,
                        color: surfaceTheme.textMuted,
                      ),
                    ),
                    const Spacer(),
                    OutlinedButton(
                      onPressed: widget.onClose,
                      child: const Text('취소'),
                    ),
                    const SizedBox(width: 10),
                    ElevatedButton(
                      onPressed: _saveDraft,
                      child: const Text('저장'),
                    ),
                  ],
                ),
              ),
            ],
          ),
        ),
      ),
    );
  }

            ),
            _SliderCard(

              description: '음성 인식 민감도를 조정합니다.',
              value: _draftSettings.general.microphoneSensitivity,

              onChanged: (value) => _update(
                _draftSettings.copyWith(

              value: _draftSettings.general.ttsSpeed,

                _draftSettings.copyWith(

          ],

      case 1:

class _Sidebar extends StatelessWidget {
  const _Sidebar({
    required this.items,
    required this.selectedTab,
    required this.onSelect,
  });

  final List<String> items;
  final int selectedTab;
  final ValueChanged<int> onSelect;

  @override
  Widget build(BuildContext context) {
    final surfaceTheme = Theme.of(context).extension<AppSurfaceTheme>()!;

    return SizedBox(
      width: 190,
      child: ListView.builder(
        padding: const EdgeInsets.all(16),
        itemCount: items.length,
        itemBuilder: (context, index) {
          final selected = selectedTab == index;

          return Padding(
            padding: const EdgeInsets.only(bottom: 8),
            child: InkWell(
              borderRadius: BorderRadius.circular(14),
              onTap: () => onSelect(index),
              child: Container(
                padding: const EdgeInsets.symmetric(horizontal: 14, vertical: 12),
                decoration: BoxDecoration(
                  color: selected ? surfaceTheme.contentBackground : Colors.transparent,
                  borderRadius: BorderRadius.circular(14),
                  border: Border.all(
                    color: selected ? surfaceTheme.accent : surfaceTheme.border,
                  ),
                ),
                child: Text(
                  items[index],
                  style: TextStyle(
                    fontFamily: 'Pretendard',
                    fontSize: 14,
                    fontWeight: selected ? FontWeight.w700 : FontWeight.w500,
                    color:
                        selected ? surfaceTheme.accent : surfaceTheme.textPrimary,
                  ),
                ),
              ),
            ),
          );
        },
      ),
    );
  }
}

3. home_page.dart는 왜 크게 안 바꾸나

lib/features/home/presentation/home_page.dart

이번 수정의 핵심은 설정 모달입니다. 현재 화면 읽기 버튼은 모달을 띄우지 않습니다. 이 버튼은 기존처럼 읽기 신호를 보내고, 결과는 현재 화면 상태 흐름 안에서 처리합니다.

정리
`현재 화면 읽기`는 음성 듣기처럼 액션 신호를 보내는 버튼입니다. 따라서 8일차에서는 home_page.dart와 demo_home_page.dart를 화면 읽기 모달 기준으로 바꾸지 않습니다.

4. demo_home_page.dart는 왜 그대로 두나

lib/demo/demo_home_page.dart

데모 모드 역시 `현재 화면 읽기`는 읽기 신호를 보내는 액션입니다. 따라서 이번 Day 8에서는 데모 홈의 화면 읽기 동작을 모달 중심으로 바꾸지 않습니다.

즉 Day 8의 실제 변경 대상은 설정 모달과 설정 컨트롤러입니다. 데모 홈은 Day 7에서 만든 설정 모달/단축키 흐름을 그대로 유지합니다.

5. 실행 후 확인할 것

  1. 실제 모드에서 설정 모달을 열고 단축키 캡처가 되는지 확인합니다.
  2. 설정 모달에서 값을 바꿔도 저장 전까지는 draft 상태로만 유지되는지 확인합니다.
  3. 저장 후 실제 테마나 단축키가 반영되는지 확인합니다.
  4. 현재 화면 읽기를 눌렀을 때 모달이 뜨지 않고 기존 액션 흐름이 유지되는지 확인합니다.
  5. 데모 모드 설정/단축키와 실제 모드 설정/단축키의 구조가 비슷해졌는지 확인합니다.

6. 오늘 꼭 이해해야 하는 것

  • 왜 실제 모드 설정 모달도 draft 상태를 가져야 하는가
  • 왜 단축키는 TextField보다 Focus + onKeyEvent가 적합한가
  • 왜 현재 화면 읽기는 결과 모달이 아니라 신호 전송 액션으로 유지해야 하는가
  • 왜 실제 모드 설정 UX를 데모 모드와 맞추는 것이 다음 단계로 중요한가

7. Day 8 완료 기준

  • 실제 모드 설정 모달이 데모 모드와 같은 구조로 동작한다.
  • 실제 모드에서 단축키 직접 캡처가 가능하다.
  • 현재 화면 읽기 버튼은 모달을 띄우지 않고 기존처럼 읽기 신호를 보낸다.
  • 실제 모드 설정 UX가 데모 모드 설정 UX와 같은 계열로 정리된다.
