Day 15

Toast System Upgrade

15일차는 앱 안에서 잠깐 뜨는 알림 UI를 하나의 공통 시스템으로 정리하는 단계입니다. 오늘은 app_toast.dart 하나만 수정하지만, 실제 모드와 데모 모드 전체의 피드백 규칙이 더 통일되고 앞으로 Windows/macOS 공통 버전을 확장하기 쉬운 형태가 됩니다.

오늘 목표: 상태 enum 확장, 문구 기반 상태 추론, DisplaySettings 반영 준비, 그리고 overlay 토스트 구조를 더 체계적인 시스템으로 정리합니다.

오늘 수정할 파일

  • lib/features/notifications/presentation/app_toast.dart

이 파일은 앱 내부 토스트를 공통으로 관리하는 presentation 파일입니다. 각 화면은 직접 토스트 UI를 만들지 않고 showAppToast(...)만 호출하고, 실제 렌더링과 상태 판단은 이 파일이 맡습니다.

왜 지금 이 파일을 정리하나

  • 실제 모드와 데모 모드가 같은 알림 규칙을 공유해야 합니다.
  • 앞으로 실패, 경고, 재시도 같은 상태도 더 명확히 나눠야 합니다.
  • 큰 글씨 같은 표시 설정을 토스트에도 반영할 준비가 필요합니다.
  • 나중에 OS 팝업 계층이 들어와도 호출부는 유지하고 내부 구현만 갈아끼울 수 있어야 합니다.

핵심 개념

  1. AppToastState 확장: error, warning를 추가해 실패 계열 피드백도 공통으로 다룹니다.
  2. 상태 추론: 호출부가 상태를 생략해도 제목과 메시지 문구로 대략의 상태를 자동 추론합니다.
  3. DisplaySettings 반영 준비: 지금은 largeText 중심으로 반영하고, 나중에 다크/고대비로 확장할 수 있게 파라미터를 받아둡니다.
  4. Palette 분리: 상태별 색상 묶음을 helper로 빼서 토스트 UI가 단순해집니다.

최종 코드

lib/features/notifications/presentation/app_toast.dart

오늘은 이 파일을 사실상 전체 교체하는 편이 가장 안전합니다. 아래 코드가 현재 Day 15 최종 상태입니다.

import 'dart:async';

import 'package:flutter/material.dart';

import '../../../shared/models/settings_models.dart';

enum AppToastState {
  info,
  listening,
  processing,
  success,
  error,
  warning,
}

void showAppToast(
  BuildContext context,
  String message, {
  String title = 'Navi: Voice Navigator',
  AppToastState? state,
  DisplaySettings? displaySettings,
  int durationMs = 2400,
}) {
  final resolvedState = state ?? _inferToastState(title, message);

  _ToastOverlay.instance.show(
    context,
    title: title,
    message: message,
    state: resolvedState,
    displaySettings: displaySettings,
    durationMs: durationMs,
  );
}

AppToastState _inferToastState(String title, String message) {
  final source = '$title $message'.toLowerCase();

  if (source.contains('실패') ||
      source.contains('오류') ||
      source.contains('연결하지 못했습니다')) {
    return AppToastState.error;
  }

  if (source.contains('경고') ||
      source.contains('재시도') ||
      source.contains('지연')) {
    return AppToastState.warning;
  }

  if (source.contains('듣고 있습니다') ||
      source.contains('음성 수신') ||
      source.contains('listening')) {
    return AppToastState.listening;
  }

  if (source.contains('처리 중') ||
      source.contains('처리하고 있습니다') ||
      source.contains('processing')) {
    return AppToastState.processing;
  }

  if (source.contains('완료') ||
      source.contains('성공') ||
      source.contains('읽어드립니다')) {
    return AppToastState.success;
  }

  return AppToastState.info;
}

class _ToastOverlay {
  _ToastOverlay._();

  static final _ToastOverlay instance = _ToastOverlay._();

  OverlayEntry? _entry;
  Timer? _dismissTimer;

  void show(
    BuildContext context, {
    required String title,
    required String message,
    required AppToastState state,
    required DisplaySettings? displaySettings,
    required int durationMs,
  }) {
    _dismissTimer?.cancel();
    _entry?.remove();

    final overlay = Overlay.of(context);
    _entry = OverlayEntry(
      builder: (context) {
        return _ToastView(
          title: title,
          message: message,
          state: state,
          displaySettings: displaySettings,
        );
      },
    );

    overlay.insert(_entry!);

    _dismissTimer = Timer(Duration(milliseconds: durationMs), () {
      _entry?.remove();
      _entry = null;
    });
  }
}

class _ToastView extends StatelessWidget {
  const _ToastView({
    required this.title,
    required this.message,
    required this.state,
    required this.displaySettings,
  });

  final String title;
  final String message;
  final AppToastState state;
  final DisplaySettings? displaySettings;

  @override
  Widget build(BuildContext context) {
    final palette = _resolvePalette(state);
    final icon = _resolveIcon(state);
    final largeText = displaySettings?.largeText ?? false;

    return Positioned(
      top: 24,
      right: 24,
      child: Material(
        color: Colors.transparent,
        child: Container(
          width: largeText ? 360 : 320,
          padding: EdgeInsets.all(largeText ? 18 : 16),
          decoration: BoxDecoration(
            color: palette.background,
            borderRadius: BorderRadius.circular(18),
            border: Border.all(color: palette.border),
            boxShadow: const [
              BoxShadow(
                color: Color(0x22000000),
                blurRadius: 18,
                offset: Offset(0, 10),
              ),
            ],
          ),
          child: Row(
            crossAxisAlignment: CrossAxisAlignment.start,
            children: [
              Container(
                width: 40,
                height: 40,
                decoration: BoxDecoration(
                  color: palette.iconBackground,
                  borderRadius: BorderRadius.circular(12),
                ),
                child: Icon(
                  icon,
                  color: palette.iconColor,
                  size: 20,
                ),
              ),
              const SizedBox(width: 12),
              Expanded(
                child: Column(
                  crossAxisAlignment: CrossAxisAlignment.start,
                  children: [
                    Text(
                      title,
                      style: TextStyle(
                        fontSize: largeText ? 15 : 14,
                        fontWeight: FontWeight.w800,
                      ),
                    ),
                    const SizedBox(height: 4),
                    Text(
                      message,
                      style: TextStyle(
                        fontSize: largeText ? 14 : 13,
                        height: 1.45,
                        color: const Color(0xFF475569),
                      ),
                    ),
                  ],
                ),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

IconData _resolveIcon(AppToastState state) {
  switch (state) {
    case AppToastState.info:
      return Icons.info_outline_rounded;
    case AppToastState.listening:
      return Icons.mic_rounded;
    case AppToastState.processing:
      return Icons.hourglass_top_rounded;
    case AppToastState.success:
      return Icons.check_rounded;
    case AppToastState.error:
      return Icons.error_outline_rounded;
    case AppToastState.warning:
      return Icons.warning_amber_rounded;
  }
}

_ToastPalette _resolvePalette(AppToastState state) {
  switch (state) {
    case AppToastState.info:
      return const _ToastPalette(
        background: Colors.white,
        border: Color(0xFFE2E8F0),
        iconBackground: Color(0xFFEFF6FF),
        iconColor: Color(0xFF2563EB),
      );
    case AppToastState.listening:
      return const _ToastPalette(
        background: Colors.white,
        border: Color(0xFFBFDBFE),
        iconBackground: Color(0xFFEFF6FF),
        iconColor: Color(0xFF2563EB),
      );
    case AppToastState.processing:
      return const _ToastPalette(
        background: Colors.white,
        border: Color(0xFFFDE68A),
        iconBackground: Color(0xFFFEF3C7),
        iconColor: Color(0xFFF59E0B),
      );
    case AppToastState.success:
      return const _ToastPalette(
        background: Colors.white,
        border: Color(0xFFBBF7D0),
        iconBackground: Color(0xFFDCFCE7),
        iconColor: Color(0xFF16A34A),
      );
    case AppToastState.error:
      return const _ToastPalette(
        background: Colors.white,
        border: Color(0xFFFECACA),
        iconBackground: Color(0xFFFEE2E2),
        iconColor: Color(0xFFDC2626),
      );
    case AppToastState.warning:
      return const _ToastPalette(
        background: Colors.white,
        border: Color(0xFFFDE68A),
        iconBackground: Color(0xFFFFF7ED),
        iconColor: Color(0xFFEA580C),
      );
  }
}

class _ToastPalette {
  const _ToastPalette({
    required this.background,
    required this.border,
    required this.iconBackground,
    required this.iconColor,
  });

  final Color background;
  final Color border;
  final Color iconBackground;
  final Color iconColor;
}

바뀐 점을 쉽게 다시 정리

  • state를 이제 nullable로 받아서, 호출부가 원하면 생략할 수 있습니다.
  • 생략되면 _inferToastState(...)가 문구를 보고 상태를 추론합니다.
  • DisplaySettings를 optional로 받아 큰 글씨일 때 폭과 폰트 크기를 키웁니다.
  • _ToastPalette와 _resolveIcon()로 상태별 표현을 분리했습니다.

15일차 완료 기준

  • app_toast.dart가 단순 토스트 위젯이 아니라 공통 토스트 시스템 역할을 한다.
  • info / listening / processing / success / error / warning 상태를 모두 표현할 수 있다.
  • 상태를 직접 넘기지 않아도 문구 기반 추론이 가능하다.
  • largeText 반영 준비가 들어가 있다.
  • 실제/데모 모드가 같은 토스트 규칙을 공유할 수 있다.
