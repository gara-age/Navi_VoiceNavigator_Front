Day 3 Tutorial

Flutter 3일차: 데모 모드 UI 만들기

이 문서는 3일차에 진행한 demo_home_page.dart 작업을 클론코딩/튜토리얼 방식으로 다시 읽을 수 있게 정리한 문서입니다.

1단계

StatelessWidget을 StatefulWidget으로 바꿉니다.

2단계

데모 전용 상태값을 추가해 화면이 상태에 따라 변하게 만듭니다.

3단계

실제 모드와 비슷한 레이아웃으로 데모 화면을 구성합니다.

4단계

Future.delayed로 데모용 상태 흐름을 만듭니다.

5단계

결과 요약과 후속 질문을 화면에 표시합니다.

오늘의 목표

  • DemoHomePage가 더 이상 임시 안내 화면이 아니어야 합니다.
  • 왼쪽 액션 패널, 상단 상태 카드, 중앙 결과 영역이 보여야 합니다.
  • 음성 듣기, 현재 화면 읽기, 텍스트 명령 입력이 동작해야 합니다.
  • 상태가 대기 → 듣는 중 → 처리중 → 완료처럼 보이게 해야 합니다.

먼저 이해할 개념

StatefulWidget

버튼 클릭에 따라 값이 바뀌는 화면을 만들 때 필요한 기본 형태입니다.

setState()

상태값이 바뀌었으니 화면을 다시 그려달라고 Flutter에 알리는 함수입니다.

Future.delayed

실제 처리처럼 보이는 시간차 흐름을 만들 때 사용합니다.

상태 기반 UI

_isBusy, _isRecording, _summary 값에 따라 화면이 바뀝니다.

공통 위젯 재사용

ActionPanel, ReadyState, StatusCard, AppTitleBar를 재사용합니다.

1. StatefulWidget으로 전환하기

lib/demo/demo_home_page.dart

데모 화면은 버튼 클릭 후 값이 계속 바뀌어야 합니다. 그래서 기존의 StatelessWidget 형태로는 부족하고, StatefulWidget으로 전환해야 합니다.

핵심 생각: 데모 화면은 정적인 설명 페이지가 아니라, 내부 상태를 가진 시뮬레이션 화면입니다.

2. 데모용 상태값 추가하기

3일차에서 추가하는 대표 상태값은 아래와 같습니다.

  • _micStatus: 상단 상태 카드에 보여줄 마이크 상태
  • _isRecording: 현재 듣는 중인지 여부
  • _isBusy: 처리 중인지 여부
  • _summary: 결과 요약
  • _followUp: 후속 질문
String _micStatus = '대기';
bool _isRecording = false;
bool _isBusy = false;
String? _summary;
String? _followUp;

3. 실제 모드와 비슷한 레이아웃 만들기

데모 모드도 실제 모드와 비슷한 구조를 가져야 사용자가 두 모드를 같은 앱 안의 흐름으로 느낄 수 있습니다.

4. 데모 흐름 만들기

핵심은 버튼을 누르면 실제 처리처럼 보이는 흐름을 만드는 것입니다.

음성 듣기 흐름

  1. 듣는 중 상태로 바뀜
  2. 잠시 후 처리중 상태로 바뀜
  3. 잠시 후 결과 요약과 후속 질문 표시

현재 화면 읽기 흐름

  1. 처리중 상태로 바뀜
  2. 잠시 후 화면 설명 결과 표시

텍스트 명령 흐름

  1. 처리중 상태로 바뀜
  2. 입력한 텍스트를 포함한 결과 문구 표시

5. 왜 _isBusy와 mounted 체크가 필요한가

_isBusy는 처리 중에 중복 클릭을 막기 위해 필요합니다. 이 값이 없으면 데모 흐름이 쉽게 꼬입니다. 

if (_isBusy) {
  return;
}

mounted는 비동기 작업이 끝날 때 화면이 아직 살아 있는지 확인하는 안전장치입니다. 

if (!mounted) {
  return;
}

최종 코드: demo_home_page.dart

lib/demo/demo_home_page.dart
import 'package:flutter/material.dart';

import '../features/home/presentation/widgets/action_panel.dart';
import '../features/home/presentation/widgets/ready_state.dart';
import '../features/home/presentation/widgets/status_card.dart';
import '../features/home/presentation/widgets/title_bar.dart';

class DemoHomePage extends StatefulWidget {
  const DemoHomePage({super.key});

  @override
  State<DemoHomePage> createState() => _DemoHomePageState();
}

class _DemoHomePageState extends State<DemoHomePage> {
  String _micStatus = '대기';
  bool _isRecording = false;
  bool _isBusy = false;
  String? _summary;
  String? _followUp;

  Future<void> _simulateListen() async {
    if (_isBusy) {
      return;
    }

    setState(() {
      _isBusy = true;
      _isRecording = true;
      _micStatus = '듣는 중';
      _summary = '음성을 듣고 있습니다. 잠시 후 자동으로 처리 단계로 넘어갑니다.';
      _followUp = null;
    });

    await Future<void>.delayed(const Duration(milliseconds: 1200));

    if (!mounted) {
      return;
    }

    setState(() {
      _micStatus = '처리중';
      _isRecording = false;
      _summary = '명령을 분석하고 실행 계획을 준비하고 있습니다.';
    });

    await Future<void>.delayed(const Duration(milliseconds: 1500));

    if (!mounted) {
      return;
    }

    setState(() {
      _isBusy = false;
      _micStatus = '대기';
      _summary = '유튜브 검색 결과를 준비했습니다.';
      _followUp = '첫 번째 결과를 재생할까요?';
    });
  }

  Future<void> _simulateScreenRead() async {
    if (_isBusy) {
      return;
    }

    setState(() {
      _isBusy = true;
      _micStatus = '처리중';
      _summary = '화면을 읽어드리고 있습니다.';
      _followUp = null;
    });

    await Future<void>.delayed(const Duration(milliseconds: 1300));

    if (!mounted) {
      return;
    }

    setState(() {
      _isBusy = false;
      _micStatus = '대기';
      _summary =
          '현재 화면은 Navi: Voice Navigator 데모 화면입니다. 왼쪽에는 주요 기능 버튼이 있고 가운데에는 준비 상태와 결과 요약이 표시됩니다.';
      _followUp = '다른 화면도 읽어드릴까요?';
    });
  }

  Future<void> _handleTextCommand(String text) async {
    if (_isBusy) {
      return;
    }

    setState(() {
      _isBusy = true;
      _micStatus = '처리중';
      _summary = '텍스트 명령을 처리하고 있습니다.';
      _followUp = null;
    });

    await Future<void>.delayed(const Duration(milliseconds: 1200));

    if (!mounted) {
      return;
    }

    setState(() {
      _isBusy = false;
      _micStatus = '대기';
      _summary = '텍스트 명령 "$text" 처리 결과입니다.';
      _followUp = '이어서 다음 작업도 진행할까요?';
    });
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: SafeArea(
        child: Column(
          children: [
            const AppTitleBar(),
            Container(
              height: 88,
              decoration: const BoxDecoration(
                border: Border(
                  bottom: BorderSide(color: Color(0xFFE5E7EB)),
                ),
              ),
              child: Row(
                children: [
                  Expanded(
                    child: StatusCard(
                      label: '마이크 상태',
                      value: _micStatus,
                      icon: Icons.mic_none_rounded,
                    ),
                  ),
                  const VerticalDivider(width: 1, thickness: 1),
                  const Expanded(
                    child: StatusCard(
                      label: '현재 모드',
                      value: '데모 모드',
                      icon: Icons.play_circle_outline_rounded,
                    ),
                  ),
                ],
              ),
            ),
            Expanded(
              child: Row(
                children: [
                  SizedBox(
                    width: 260,
                    child: ActionPanel(
                      isRecording: _isRecording,
                      onListenPressed: _simulateListen,
                      onScreenReadPressed: _simulateScreenRead,
                      onSettingsPressed: () {},
                    ),
                  ),
                  Expanded(
                    child: ReadyState(
                      summary: _summary,
                      followUp: _followUp,
                      isBusy: _isBusy,
                      onSubmitText: _handleTextCommand,
                    ),
                  ),
                ],
              ),
            ),
          ],
        ),
      ),
    );
  }
}

3일차 완료 기준

화면 기준

  • DemoHomePage가 더 이상 플레이스홀더가 아닙니다.
  • 상단 상태 카드, 왼쪽 액션 패널, 중앙 결과 영역이 보입니다.

동작 기준

  • 음성 듣기 버튼이 상태 흐름을 보여줍니다.
  • 현재 화면 읽기 버튼이 결과를 출력합니다.
  • 텍스트 명령 입력이 결과 요약에 반영됩니다.

학습 기준

  • StatefulWidget을 왜 쓰는지 설명할 수 있습니다.
  • setState()가 언제 필요한지 이해합니다.
  • Future.delayed가 데모 흐름에 왜 적합한지 설명할 수 있습니다.
